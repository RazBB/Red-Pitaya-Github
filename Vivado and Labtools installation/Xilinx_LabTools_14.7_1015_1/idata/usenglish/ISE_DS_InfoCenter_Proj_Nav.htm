Key New Features in Project Navigator

  • Embedded Development Kit (EDK) Integration Improvements
    • Support for multiple ELF files and automatic detection of ELF files referenced by EDK designs – More Info
    • Ability to control associations between ELF files and specific processors defined in XMP files – More Info
    • Ability to select an Evaluation Development Board in the New Project Wizard, New Project dialog box, or Design Properties dialog box – More Info
    • Ability to export hardware design before running implementation – More Info
    • Ability to automatically launch the Software Development Kit (SDK) after design export – More Info
  • Project Comparison Enhancements
    • Compare Projects feature includes additional categories and layout improvements – More Info
  • SmartXplorer Enhancements
    • Support for power-dedicated and custom strategies – More Info
  • CORE Generator™ Software Enhancements
    • Ability to update a core to the latest version – More Info
    • Ability to check all core versions – More Info
  • System Generator Enhancements
    • Ability to create a new System Generator source in Project Navigator – More Info
  • Timing Analyzer Enhancements
    • Support for viewing TWR reports – More Info
